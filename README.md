#  Smart Crop Advisory System

A machine learning-powered web application that recommends the most suitable crop to grow based on soil parameters and real-time weather conditions.

---

##  Features

- **Manual Recommendation** — Enter soil parameters (N, P, K, pH, temperature, humidity, rainfall) and get an instant crop recommendation
- **Weather-Based Recommendation** — Enter your city name and soil nutrients; the app fetches live weather data automatically via OpenWeatherMap API
- **Top-3 Crop Predictions** — Each result shows the top 3 recommended crops along with confidence percentages
- **Detailed Crop Information** — View growing tips, ideal conditions, expected yield, common pests, and fertilizer requirements for 22 crops
- **ML Visualizations** — Confusion matrix and feature importance charts generated during model training

##  Setup & Installation

### 1. Clone the repository
```bash
git clone https://github.com/your-username/smart-crop-advisory.git
cd smart-crop-advisory
```

### 2. Install dependencies
```bash
pip install flask scikit-learn pandas numpy matplotlib seaborn requests
```

### 3. Get your OpenWeatherMap API key
- Sign up for free at [openweathermap.org](https://openweathermap.org/api)
- Copy your API key
- Open `config.py` and replace the placeholder:
```python
WEATHER_API_KEY = 'your_actual_api_key_here'
```

### 4. Train the model
```bash
python model_training.py
```
This will generate `models/crop_model.pkl`, `models/scaler.pkl`, and save visualizations to `static/`.

### 5. Run the application
```bash
python app.py
```
Open your browser and go to: **http://127.0.0.1:5000**

##  How the ML Model Works

- **Dataset:** Kaggle Crop Recommendation Dataset — 2,200 records, 22 crop types
- **Features:** Nitrogen (N), Phosphorus (P), Potassium (K), Temperature, Humidity, pH, Rainfall
- **Algorithm:** Random Forest Classifier (100 estimators, max depth 15)
- **Preprocessing:** StandardScaler for feature normalization
- **Split:** 80% training / 20% testing (stratified)
- **Output:** Predicted crop + probability scores for all 22 classes
